Chromecast Webpage Viewer

Cast webpages to a Chromecast.

Requires Chrome with the Google Cast plugin

Insert a URL in the box below and click 'Cast It', this will open the Chromecast plugin to open, choose a Chromecast from this list. This should then display the webpage on the selected Chromecast.

This is acheived using an iframe, some webpages (for example google.com & twitter.com) are set to disable being displayed in this manor.
